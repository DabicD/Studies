Project created with cooperation with my friend to pass QT subject.

Project contains 2 working tabs.

Spam-tool tab:
You can choose activation key, delay between spaming and the file with different messages in every line.
After clicking Start button your activation key will be available.

lol-tool tab:
You can create your own messages and bind it to selected keys, your binds will be shown right below.
Main function of this tool is bot-script that allows you to hold the chosen key and make perfect kite*.
Works with localdata.

lol - League of Legends
kite - This is running in time between gaps of your attacks.
(More attack-speed = faster clicking, easy to make a mistake)
